Comandos para Git

    git init 
// Inicializar tu repositorio local
  • git add
    • Agregar cambios a stage de git, siempre se tiene que hacer git add al archivo antes del commit
  • git commit -m
    • Mandar todos tus cambios al repositorio local
  • git push
    • Mandar tu repositorio local a repositorio remoto
  • git log
    • checar los commits que se han hecho a el archivo o la rama completa
  • git fetch
    • jalar cambios de repositorio remoto a local, pero es mejor usar pull
  • git pull origin master
    • Tener la ultima version del repositorio remoto
  • git push origin master
    • Empujar a origin (remote) la rama master
  • git merge
    • combinar dos ramas distintas
  • git remote -v
    • conocer si hay repositorio remoto
  • git branch
    • ver las ramas que existen
  • git switch -c "nombre de nueva branch"
    • crear una rama nueva y brincar a ella
  • git checkout
    • cambiar entre ramas
